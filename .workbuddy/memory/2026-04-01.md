## 下午补充：Halo博客系统源码学习（16:02）

### 项目概述
- **项目位置**：`/Users/derek/Desktop/MyFile/毕业论文/halo/`
- **技术栈**：Java (Spring Boot) + Vue.js + MySQL/H2
- **架构**：前后端分离，B/S架构
- **许可证**：GPLv3 开源

### 核心模块结构

**后端 (application/ + api/)**
```
run/halo/app/
├── content/          # 内容管理：文章、页面、分类、标签、评论
│   ├── impl/         # 内容服务实现
│   └── comment/      # 评论服务
├── core/             # 核心功能
│   ├── attachment/   # 附件管理
│   ├── endpoint/     # API端点
│   ├── reconciler/   # 状态协调器
│   └── user/         # 用户管理
├── extension/        # 扩展机制（核心框架）
│   ├── index/        # 索引机制
│   └── store/        # 扩展存储
├── infra/            # 基础设施
├── notification/     # 通知机制
├── plugin/           # 插件系统
├── security/         # 安全认证
└── theme/            # 主题系统
```

**前端 (ui/)**
```
ui/
├── console-src/      # 管理后台 (Vue.js)
├── uc-src/           # 用户中心
├── packages/         # 共享包
│   ├── api-client/   # API客户端
│   ├── components/   # 组件库
│   ├── editor/       # 富文本编辑器
│   └── console-shared/# 控制台共享
└── src/              # 主题端
```

### 核心技术特性

**1. 扩展机制 (Extension)**
- 基于Kubernetes的Extension模型
- 使用GVK (GroupVersionKind) 定义资源类型
- 支持插件热加载和动态扩展

**2. 索引机制 (Index)**
- 解决大数据量查询性能问题
- 支持自定义索引字段
- 内存索引 + 可选磁盘持久化

**3. 内容模型**
- Post (文章)：标题、别名、摘要、分类、标签、可见性
- Snapshot (快照)：版本控制，支持多版本管理
- Category (分类)：支持层级结构
- Tag (标签)：支持颜色和封面
- Comment/Reply (评论/回复)：支持嵌套

**4. 插件扩展点**
- ReactivePostContentHandler：文章内容处理
- ReactiveSinglePageContentHandler：页面内容处理
- 认证扩展点：FormLogin、Authentication、Anonymous

**5. 全文搜索**
- 支持多种搜索引擎：Lucene、ElasticSearch、MeiliSearch
- 中文分词支持（jieba、ik-analyzer等）
- 高亮搜索结果

### 数据库设计要点
- 使用JPA/Hibernate作为ORM
- Extension存储为JSON格式
- 支持MySQL、PostgreSQL、H2
